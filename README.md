## Development

Building WPPConnect is really simple altough it contains 3 main projects inside

### 1. Wapi project

```bash
> npm run build:wapi
```

### 2. Middleware

```bash
> npm run build:middleware
> npm run build:jsQR
```

### 3. Wppconnect

```bash
> npm run build:client
```

To build the entire project just run

```bash
> npm run build
```
